- 功能
- 智能标签生成：领域 + 类型 + 技术 + 来源四维标签体系
- 摘要提取：一句话抓住文章核心
- 去重检测：标题相似度 0.3 + 内容相似度 0.7，阈值 0.7
- 关联推荐：自动关联已有笔记
- 元数据标准化：YAML frontmatter 格式
- 多来源支持：微信公众号、网页等


- 使用方法
- 用户：存入知识库 https://mp.weixin.qq.com/s/xxx

- OpenClaw：
- 1. 自动提取文章内容
- 2. 检查是否已存在
- 3. 生成标签和摘要
- 4. 输出 Markdown 笔记

- 适用场景
- 定期整理微信收藏的文章
- 建立个人知识库体系
- 团队知识沉淀
- 学术资料归档

OpenClaw-compatible skill. Codex-compatible skill.
